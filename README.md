# Inatro_usecase_app
Aplicativo para consultas de matriculas de veiculos

## As tecnologias usadas:
### Flutter
### Dart
### Firebase
- Authentication
- Cloud Firestore

## Features:
- Login
- Cadastro
- Atualizar Dados do utilizador
- Fazer Pagamento (feature ainda em manutenção)
- Fazer consulta de matriculas de veiculos
